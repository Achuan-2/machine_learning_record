●关于附录文件
附录文件是《用Python动手学机器学习》中正文解说的代码和运行结果的源码文件（从第1章到第9章）。
源码文件为Jupyter Notebook格式。
关于Jupyter Notebook的安装方法，请参考本书第1章。

・向Jupyter Notebook添加源码文件的方法
Jupyter Notebook启动后，将会打开Files选项卡（显示在Windows的浏览器上）。
将原文文件拖拉到［Files］选项卡的界面上即可完成添加。
添加后点击［Upload］按钮。
点击上传后的文件打开编辑界面。
你可以在［In］单元格复制代码，也可以在文件上查看运行结果。

●本书源码的测试环境 
　本书源码的测试环境如下所示，已确认源码可正常运行。

OS：Windows 10

Anaconda：4.6.11
※本书使用的「Anaconda3-2019.03-Windows-x86_64.exe」可从 https://repo.anaconda.com/archive/ 下载。
Python：3.7.3
Jupyter Notebook：5.7.8
TensorFlow：1.13.1
Keras：2.2.4

●安装指定版本的方法
在安装上面指定的TensorFlow和Keras的版本时，请参考本书第1.4节的介绍，在安装命令中指定版本号安装。

pip install tensorflow==1.13.1

pip install keras==2.2.4
